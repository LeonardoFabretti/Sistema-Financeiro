## 📱 Como Usar:

1. **Gerar os Ícones:**
   - Abra `public/icon-generator.html` no navegador
   - Clique em "Download Todos"
   - Salve os ícones na pasta `public/`

2. **Testar Localmente:**
   ```bash
   npm run dev
   ```

3. **Instalar o App:**
   - No celular: Abra no navegador e clique em "Adicionar à tela inicial"
   - No PC: Clique no ícone de instalação na barra de endereços

4. **Deploy Online:**
   - Vercel: `vercel`
   - Netlify: `netlify deploy --prod`
   - Ou qualquer outro serviço de hosting
